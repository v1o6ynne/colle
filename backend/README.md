## Colla: Co-Creating An Accessible Online Paper Reading Community

### Setup
1. If you do not have Node.js, [install it from here](https://nodejs.org/en/)

2. Make sure you use it on chrome

3. Clone this repository.

4. Navigate into the project directory
`$ cd colla`

5. Activate the local server
`node server.js`

6. Load the project through chrome extension by unpacking the root folder

![image](https://github.com/ucla-hci/colla/assets/72054295/002de4b2-396e-401d-bdf1-cdfcff67e426)

7. You should be able to see the app on the sidepanel of Chrome
